# Customer Churn Prediction 🚀

## 📌 Project Overview
This project predicts whether a customer will leave a telecom company or stay using Machine Learning.

## 📊 Problem Statement
Customer churn is a major issue for subscription-based companies. The goal is to identify customers likely to leave.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📁 Dataset
Telco Customer Churn Dataset (Kaggle)

## 🔍 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Model Building (Logistic Regression)
- Model Evaluation

## 🤖 Machine Learning Model
- Logistic Regression

## 📈 Result
Achieved good accuracy in predicting customer churn.

## 📌 Outcome
Helps businesses understand customer behavior and reduce churn rate.
